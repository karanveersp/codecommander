Python Resources

Sun Sep 29, 2019 by Karanveer in  python, resources

Learning python is easier than other languages for many reasons. One of the reasons is the tremendous amount of great, free resources to learn from. So here are some resources I found heplful.

Books

  • Automate the Boring Stuff with Python - Shows how to write useful programs to do all sorts of things with python.
  • Composing Programs - A python oriented take on the legendary SICP book/course.
  • Think Python: How to Think Like a Computer Scientist - Get immersed in the details.
  • Learn Python - Fastest introduction to python if you have programmed in other languages.

Videos

  • Corey Schafer’s Beginning Python Programming Playlist - Python centric channel that has lots of quality content from basics to advanced concepts.
  • Derek Banas’s Learn Python in One Video - Efficient and to the point.

I also learned quite a bit about Python and REST API development through this amazing Udemy course. Highly recommend you grab it on one of the periodic sales.
